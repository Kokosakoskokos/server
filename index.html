Tutorial

Use Python + FastAPI to serve system metrics and actions.

# Install FastAPI + Uvicorn
pip3 install fastapi uvicorn psutil

# Example app (backend/main.py)
from fastapi import FastAPI
import psutil, subprocess

app = FastAPI()

@app.get("/stats")
def stats():
    return {
        "cpu": psutil.cpu_percent(),
        "ram": psutil.virtual_memory().percent,
        "uptime": psutil.boot_time()
    }

@app.post("/power/on")
def power_on():
    subprocess.run(["racadm", "-r", "IDRAC-IP", "-u", "root", "-p", "password", "serveraction", "powerup"])
    return {"status": "powering on"}

Run backend: uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000

Tutorial

Use React (or plain HTML/JS) to show system stats and buttons.

npx create-react-app dashboard
cd dashboard
npm start

In React, fetch backend API:

useEffect(() => {
  fetch("http://SERVER-IP:8000/stats")
    .then(res => res.json())
    .then(data => setStats(data));
}, []);

Add buttons to call /power/on, /power/off, etc.

Tutorial

Use backend API to toggle smart plug and server power.

# Example FastAPI endpoint
@app.post("/power/on-full")
def on_full():
    requests.post("http://SMARTPLUG-IP/on")
    subprocess.run(["racadm", "-r", "IDRAC-IP", "-u", "root", "-p", "pw", "serveraction", "powerup"])
    return {"status": "SmartPlug+Server On"}

On shutdown: call server poweroff → wait → smart plug off.
